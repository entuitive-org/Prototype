Functions as Graphs

Graphs are among the most powerful tools we have in mathematics. We can use them to represent functions, either from formulas or directly. We'll cover formulas in the next section.

Most often, the independent variable (usually \(x\)) is draw on the horizontal axis, and the dependent variable (usually \(y\)) on the vertical axis.

Canvas don't work on your computer.

This graph of the line \( y = \frac{1}{2} x + 1 \) has a domain and range of all real numbers, \( \mathbb{R} \), or \( ( -\infty, \infty ) \).

An arrowhead at the end of a line indicates that it continues indefinitely in that direction.

To get the function value for a certain input, you can draw (or imagine) a vertical line at that point. The height of the graph at that point, \(a\), is your function value, \( f(a) \).

If the line intersects at more than one place, then the graph does NOT represent a function. This is known as the vertical line test.

\( f(a) = b \)

\( f(a) = b \)

As with a table, a single graph can include multiple function plots. You can treat each graph as being separate from the others, but sometimes it is helpful to visualize them together.

\( f(a) = b \)

\( g(a) = c \)

\( h(a) = d \)
