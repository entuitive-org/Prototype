Functions as Tables

Functions can be represented by tables, with inputs in one column or row and the outputs in another:

vertical// IN/ OUT// 1/ 2// 3/ 4// 6/ 7//
horizontal// IN/ 1/ 3/ 6// OUT/ 2/ 4/ 7//

The input or independent variable is often denoted \(x\), and the function \(f\) or \(f(x)\), pronounced "\(f\) of \(x\)". The output of \(f\) may be assigned to a dependent variable, such as \(y\), i.e. \(y=f(x)\).

horizontal// x/ 1/ 2/ 3/ 4// f(x)/ 1/ 4/ 9/ 16//
vertical// x/ y// 2/ 4// 3/ 9// 4/ 16//

We can name our variables and functions whatever we like. We can also have multiple functions for a single independent variable in the same table. Here is an example where the independent variable is \(t\) with three functions, \(d\), \(v\), and \(a\):

horizontal// t/ 0/ 1/ 2/ 3/ 4/ 5// d(t)/ 0/ 5/ 20/ 45/ 80/ 125// v(t)/ 0/ 10/ 20/ 30/ 40/ 50// a(t)/ 10/ 10/ 10/ 10/ 10/ 10//

To find the output for a given input value, you can read it from the corresponding row/column of the table. For example, in the table above, \(d(1)=5\), \(v(3)=30\), and \(a(5)=10\). Try it for yourself:
